Usage Instructions

How to Use This Library

  1. Copy Component HTML: Find the component you need and copy the entire HTML block
  2. Paste into Page: Insert into your static HTML page where needed
  3. Customize Content: Replace placeholder text with actual content
  4. Update Links: Change href values to match your site structure
  5. Test Responsive: Verify on mobile, tablet, and desktop

Brand System Compliance

  • All components use Deep Navy (#0A0F1F) for dark sections
  • Electric Blue → Purple gradients for CTAs and accents
  • Navigation motifs (topographic, grid, route lines) in backgrounds
  • Consulting-quality typography (bold headings, clean body)
  • Every section includes visual elements (no plain text sections)
